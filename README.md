# Amazon Homepage Clone 🛒

A front-end clone of the Amazon homepage built using **HTML and CSS**.

This project was created as part of my web development learning journey to practice **page layout, styling, navigation bars, sections, and responsive design concepts**.

## 🚀 Features

* Amazon-style navigation bar
* Search bar section
* Product showcase sections
* Header and footer layout
* Styled using pure CSS
* Responsive front-end structure
* Image assets for product sections and banners

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**

## 📚 What I Learned

Through this project, I practiced:

* HTML page structure
* semantic tags
* `div` layouting
* navigation bar design
* CSS positioning
* box model
* flexbox basics
* styling images and sections
* footer and header creation

## 📁 Project Files

* `amazonclone.html` → main HTML file
* `amazonclone.css` → styling file
* other files → image assets used in the project
